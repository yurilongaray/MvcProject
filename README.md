# MvcProject
O problema
Seu objetivo é desenvolver uma solução WEB que realize a inscrição em um curso
profissionalizante para pessoas acima de 18 anos. O usuário deverá visualizar os cursos
disponíveis e então preencher um formulário de cadastro para finalizar a sua inscrição:
Seção: Cursos Disponíveis
? Gastronomia
? Enfermagem
? Farmácia
? Desenvolvimento de Software
? Administração
Para cada curso, deverá ter as informações de carga horária, data de início, dias da
semana que ocorrerão e um vídeo explicativo do curso (qualquer vídeo do Youtube).
Ao selecionar o curso, o usuário deverá preencher um formulário contendo os seguintes
campos:
? Nome (obrigatório)
? E-mail (obrigatório)
? Data de Nascimento (obrigatório)
? Telefone (opcional)
Ao confirmar o preenchimento do formulário, deverá ser apresentado uma mensagem
específica para o curso selecionado, onde a mensagem em questão deverá ter em seu
conteúdo algo que o identifique.
