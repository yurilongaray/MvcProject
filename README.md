# MvcProject
O problema
Seu objetivo é desenvolver uma solução WEB que realize a inscrição em um curso
profissionalizante para pessoas acima de 18 anos. O usuário deverá visualizar os cursos
disponíveis e então preencher um formulário de cadastro para finalizar a sua inscrição.

Obs.:
1. A presente aplicação foi desenvolvida utilizando o Visual Studio CODE não o Visual Studio convencional;
2. O input de data de nascimento está com formato (visual) incorreto porém no backend está sendo possível receber seu resultado e validar o mesmo;

Melhorias:
1. Caso possuisse mais tempo, modificaria o DateType do model de para String e dessa forma trataria a inserção das datas por JQuery e, com isso, utilizaria um split para pegar somente a data e validar a data com a idade do aluno.
